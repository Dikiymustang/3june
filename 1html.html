Создание репозитория

  • создаём на git hub новый репозиторий.
  • создаём папку на компьютере с тем же именем.
  • в консоли встаём на эту папку и задаём команды:

    git init

    создаем файлы в своей папке

    git add .

    git commit -m "какой-нибудь комментарий"

    git branch -M main (если нужна другая ветка, не master)

    git remote add origin адрес_на_гитхабе

    git push --set-upstream origin master
